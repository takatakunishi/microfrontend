# What is this project

このレポジトリはマイクロフロントエンドのコンテナアプリケーションです。

## 使用方法

- `make up`: コンテナの立ち上げ(<http://localhost:3100/>)
- `make down`: コンテナを終了する
- `make preview`: buildとpreviewを実行
  - Web Component(Shadow DOM)用 <http://localhost:3150/>
