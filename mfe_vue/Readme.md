# What is this project

このレポジトリはマイクロフロントエンドのVue(3)サービスです。

## 使用方法

- `make up`: コンテナの立ち上げ(http://localhost:3103/)
- `make down`: コンテナを終了する

mfe_vueコンテナ内限定

- `make preview`: buildとpreviewを実行(http://localhost:3153/)
