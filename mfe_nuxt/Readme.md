# What is this project

このレポジトリはマイクロフロントエンドのNuxt3サービスです。

## 使用方法

- `make up`: コンテナの立ち上げ(<http://localhost:3102/>)
- `make down`: コンテナを終了する
- `make preview`: buildとpreviewを実行
  - <http://localhost:3153/runtime.html>

## 条件

- Docker (docker compose)が利用可能
- Makeが利用可能 (なくても良い)

## 起動確認

- m1 Mac
